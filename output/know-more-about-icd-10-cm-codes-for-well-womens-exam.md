ICD-10-CM Codes for Well Women’s Exam
-------------------------------------

Commonly used ICD-10 CM codes for [well women’s exam](https://www.medicalbillersandcoders.com/article/cpt-coding-for-well-woman-exams.html) include Z01.411; Z01.419; Z01.411; Z01.419; Z12.4; Z12.72; and Z13.6

**1\. Z01.411: Encounter for gynecological examination (general) (routine) with abnormal findings**

This code is used when a gynecological examination is performed as a routine check-up, and abnormal findings are discovered. Use this code when the purpose of the visit is a routine gynecological examination, and there are abnormal findings in the patient’s reproductive or pelvic organs or other gynecological conditions. Ensure that the abnormal findings are documented clearly in the medical record to support the use of this code. The encounter is typically billed as an outpatient visit or preventive examination, depending on the setting.

**2\. Z01.419: Encounter for gynecological examination (general) (routine) without abnormal findings**

This code is used when a gynecological examination is performed as a routine check-up without any abnormal findings. Use this code when the purpose of the visit is a routine gynecological examination and no abnormal findings or gynecological conditions are present. Ensure that the absence of the abnormal conclusions is documented clearly in the medical record to support the use of this code. The encounter is typically billed as an outpatient visit or preventive examination, depending on the setting.

**3\. Z01.411: Encounter for cervical Pap smear following the abnormal result of cervical Pap smear on the previous examination**

This code is used when a follow-up cervical Pap smear is performed after an abnormal result on a previous Pap smear. Use this code when the purpose of the encounter is a follow-up Pap smear due to an abnormal result from a previous Pap smear. Ensure that the abnormal result from the previous Pap smear is clearly documented in the medical record to support the use of this code. The encounter is typically billed as an outpatient visit or preventive examination, depending on the setting.

**4\. Z01.419: Encounter for cervical Pap smear following the typical result of cervical Pap smear on the previous examination**

This code is used when a follow-up cervical Pap smear is performed after a previous Pap smear with a typical result. Use this code when the purpose of the encounter is a follow-up Pap smear, specifically following a previously typical Pap smear result. Ensure that the normal consequence of the previous Pap smear is clearly documented in the medical record to support the use of this code. The encounter is typically billed as an outpatient visit or preventive examination, depending on the setting.

**5\. Z12.4: Encounter for screening for malignant neoplasm of the cervix**

This code is used when an encounter is solely for screening for cervical cancer. Use this code when the primary objective of the visit is the screening for cervical cancer, such as through a Pap smear or HPV testing. Ensure that the screening test or procedure performed is documented clearly in the medical record to support the use of this code. The encounter is typically billed as a preventive service, such as a preventive office visit or preventive examination.

**6\. Z12.72: Encounter for screening for malignant neoplasm of breast**

This code is used when an encounter is solely for screening for breast cancer. Use this code when the primary objective of the visit is the screening for breast cancer, such as through a mammogram or clinical breast examination. Ensure that the screening test or procedure performed is documented clearly in the medical record to support the use of this code. The encounter is typically billed as a preventive service, such as a preventive office visit or preventive examination.

**7\. Z13.6: Encounter for screening for osteoporosis**

This code is used when an encounter is solely for osteoporosis screening. Use this code when the primary objective of the visit is the screening for osteoporosis, such as through bone density testing. Ensure that the screening test or procedure performed is documented clearly in the medical record to support the use of this code. The encounter is typically billed as a preventive service, such as a preventive office visit or preventive examination.
